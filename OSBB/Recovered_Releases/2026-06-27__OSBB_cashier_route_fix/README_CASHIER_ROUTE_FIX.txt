OSBB — исправление маршрута кассы
=================================

Проблема
--------
После установки телефонной ветки кнопка «💰 Касса» отвечала только:
    «Вы нажали: 💰 Касса»

Причина: обработчик клиентского кабинета перехватывал текст кнопки
раньше кассы v2.

Исправление
-----------
Этот файл заменяет только launcher:
    run_bot_live_services_sandbox_v1.py

При старте launcher переставляет в памяти порядок обработчиков:
    1. касса v2;
    2. клиентский кабинет.

Bots\parking_bot.py не изменяется.
База sandbox не изменяется этой заменой.

Установка
----------
1. Остановите bot через Ctrl+C.
2. Скопируйте run_bot_live_services_sandbox_v1.py из этого архива в:
     G:\Programming\Py\OSBB\
   с заменой прежнего файла.
3. Запустите:
     Start_OSBB_Live_Services_Sandbox_Bot_v1.bat

В окне запуска должна появиться строка:
    - cashier route: placed before client portal
или:
    - cashier route: already before client portal

После этого:
Главное меню → Админ-режим → 💰 Касса
→ 🔔 Уведомления жителей
→ N-20260627172646-000003
→ ✅ Подтвердить наличные
→ в заметке введите: -
