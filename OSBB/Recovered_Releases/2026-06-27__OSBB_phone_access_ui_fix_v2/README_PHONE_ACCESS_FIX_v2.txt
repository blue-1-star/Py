        OSBB — исправление телефонного доступа, установщик v2
        ======================================================

        Предыдущий .bat использовал обычную команду COPY. На данном компьютере
        Windows отказался заменить файл. Этот вариант не использует COPY.

        Установка
        ----------
        1. В окне live-services sandbox bot нажмите Ctrl+C.
        2. Распакуйте этот архив прямо в:
             G:\Programming\Py\OSBB\
        3. Дважды нажмите:
             INSTALL_PHONE_ACCESS_UI_FIX_v2.bat

        Установщик заменяет только:
             Bots\handlers\service_orders_workspace.py

        Перед заменой он:
        - создаёт резервную копию исходного файла;
        - создаёт временный файл рядом с целью;
        - проверяет его синтаксис тем же Python из venv;
        - снимает атрибут Read-only, если он установлен;
        - заменяет исходный файл атомарно.

        Если Windows всё ещё откажет в доступе, окно сообщит точную причину.
        Тогда нужно убедиться, что bot остановлен и файл
        service_orders_workspace.py не открыт для редактирования в другой
        программе.
