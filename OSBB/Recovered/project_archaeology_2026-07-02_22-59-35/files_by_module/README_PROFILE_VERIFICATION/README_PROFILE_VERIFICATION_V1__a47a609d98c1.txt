OSBB — приветствие и проверка данных жильца (sandbox V1)
=========================================================

Назначение
----------
Перед подключением телефонного доступа житель проверяет данные в кабинете.
Система показывает квартиру, автомобили, государственные номера, марку/модель,
цвет и parking_time. Отсутствующие критичные поля выделяются отдельно.

Телефонный доступ блокируется, пока не завершены:
- привязка квартиры;
- подтверждение «автомобиля нет» при отсутствии записей авто;
- государственный номер и parking_time для каждой имеющейся машины;
- подтверждение жителем текущей карточки;
- все ожидающие операторские заявки на исправление.

Марка/модель и цвет показываются для проверки, но в V1 являются
предупреждениями, а не блокировкой. Это хранится в версии политики и может
быть изменено позднее.

Отдельный номер доступа
-----------------------
Номер, подключаемый к шлагбауму, является отдельным непубличным реквизитом.
Он может не совпадать с Telegram/контактными номерами жильца. Модуль никогда
не ищет и не требует такого совпадения, а также не меняет контакты жильца.

Что добавляет модуль
--------------------
- одноразовое приветствие после входа в клиентский режим;
- кнопка «📋 Проверить мои данные»;
- блокировка входа в «📞 Открытие телефоном» при незавершённой проверке;
- заявление Day / Night / Не паркуется для пустого parking_time;
- очередь администратора «📋 Проверка данных жителей»;
- отдельные статусы каждого выбранного шлагбаума в карточках телефонного доступа.

Установка
---------
1. Остановите sandbox bot: Ctrl+C.
2. Распакуйте архив в:

   G:\Programming\Py\OSBB\

3. Запустите:

   RUN_INSTALL_profile_verification_v1.bat

   На этом шаге база не меняется; создаются резервные копии исходников.

4. Запустите:

   RUN_MIGRATE_profile_verification_sandbox.bat

   Изменяется только live-services sandbox. Перед этим создаётся backup базы.

5. Запустите:

   RUN_CHECK_profile_verification_sandbox.bat

   Ожидаемый конец: CHECK COMPLETED

6. Запустите:

   Start_OSBB_Live_Services_Sandbox_Bot_v1.bat

Первый тест
-----------
В клиентском меню нажмите «📋 Проверить мои данные». Затем попробуйте
«📞 Открытие телефоном»: при неполном профиле будет предупреждение и переход
к проверке данных, а новый номер доступа вводить ещё не потребуется.

Не затрагивает
-------------
- osbb.db;
- существующие заявки, платежи и активные подписки;
- реальные контроллеры/шлагбаумы;
- контактные номера жильцов.
