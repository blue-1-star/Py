OSBB — изолированный операторский TEST для parking_time (квартира 40)
========================================================================

Зачем нужен этот модуль
-----------------------
Квартира 40 найдена как подходящий пример:

    квартира 40
    автомобиль AA0667HB — GRANDIS
    parking_time пустой

Нужно проверить выбор Day / Night / «Не користується паркуванням»,
операторскую очередь и логику блокировки доступа, но НЕ создавать следов,
будто житель сам открывал кабинет.

Принцип TEST
------------
TEST использует реальные данные только как снимок для экрана. Он НЕ:

- создаёт resident_profile_verifications;
- меняет READY / NEEDS_REVIEW жильца;
- создаёт welcome_shown_at;
- создаёт заявку от имени жильца;
- пишет в vehicles.parking_time;
- создаёт телефонный доступ, заказ или платёж;
- отправляет жителю Telegram-сообщение.

Единственные новые записи находятся в отдельных TEST-таблицах:

    profile_parking_time_test_sessions
    profile_parking_time_test_events

Это честный служебный аудит теста, а не след в профиле жильца.

Что проверяется
---------------
1. Исходное состояние:
   parking_time пустой → TEST-сценарий показывает:
   «телефонный доступ заблокирован».

2. Оператор в TEST выбирает:
   Day / Night / Не користується паркуванням.

3. Пока TEST ожидает решения оператора:
   «телефонный доступ заблокирован — ожидается решение оператора».

4. Оператор подтверждает TEST:
   данные квартиры 40 НЕ меняются.
   Экран честно показывает, что реальный доступ всё ещё заблокирован, потому
   что исходная база остаётся без parking_time.

Таким образом это тест UI и бизнес-правила без риска изменить квартиру 40.
Отдельное реальное исправление можно будет внести позднее через штатную
операторскую процедуру, уже с подтверждённым основанием.

Установка
---------
1. Остановите sandbox bot: Ctrl+C.

2. Распакуйте архив в:

   G:\Programming\Py\OSBB\

3. Запустите:

   RUN_INSTALL_profile_parking_time_test_v1.bat

   Здесь меняются только Python-файлы; база не открывается.

4. Запустите:

   RUN_MIGRATE_profile_parking_time_test_sandbox.bat

   Миграция создаёт резервную копию sandbox-базы и добавляет ТОЛЬКО
   отдельные TEST-таблицы.

5. Запустите:

   RUN_CHECK_profile_parking_time_test_sandbox.bat

Ожидаемый итог:

   CHECK COMPLETED

6. Запустите штатный bot:

   Start_OSBB_Live_Services_Sandbox_Bot_v1.bat

Как пройти тест
---------------
В Telegram:

    Адмін-режим
    → 🧪 Тест parking_time
    → 🧪 Відкрити TEST кв. 40
    → выбрать Day / Night / Не користується паркуванням
    → 📋 TEST-черга parking_time
    → открыть созданную TEST-сессию
    → ✅ Підтвердити TEST без зміни даних

На последней карточке должно быть видно, что:
- TEST подтвержден;
- исходный vehicles.parking_time не менялся;
- реальный профиль/телефонный доступ не затронут;
- при реальном применении ещё требовалось бы подтверждение жильца.

Важно
-----
Не использовать клиентский режим, «Изменить квартиру» или «Проверить мои
данные» для квартиры 40.
