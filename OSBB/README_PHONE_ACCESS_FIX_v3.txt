        OSBB — телефонный доступ, исправление v3
        =========================================

        Исправлена причина пустого экрана.

        Ранее экран искал услугу телефонного доступа только по одному точному
        техническому имени workflow-профиля. В фактическом sandbox-каталоге
        активная услуга имеет другое имя профиля, хотя относится к категории
        ACCESS. Поэтому текст экрана появлялся, а кнопка отсутствовала.

        Теперь:
        - телефонная услуга определяется по категории ACCESS;
        - показывается ровно одна понятная кнопка:
              📱 Підключити телефон
        - технические / тестовые дубликаты скрыты;
        - даже если Telegram временно не покажет клавиатуру, введённый номер
          на этом экране будет принят как переход к карточке намерения.

        Установка
        ----------
        1. Остановить bot: Ctrl+C.
        2. Распаковать архив в:
             G:\Programming\Py\OSBB\
        3. Запустить:
             INSTALL_PHONE_ACCESS_UI_FIX_v3.bat
        4. После SUCCESS запустить:
             Start_OSBB_Live_Services_Sandbox_Bot_v1.bat

        База данных не изменяется. Завершённый тест новых пультов не затрагивается.
