# Do not upload to the Business Project

- `G:\Prog_secret\telegram_osbb.py`
- Telegram bot tokens, passwords, API keys, `.env`
- any SQLite `.db` or `.db-wal` / `.db-shm` files
- raw resident registry or full telephone/payment spreadsheet
- database backups
- screenshots that show unneeded personal information

For debugging, use the smallest relevant code file, sanitized traceback, or a
single redacted screenshot. The handoff file is designed to carry project
context without carrying personal data or secrets.
