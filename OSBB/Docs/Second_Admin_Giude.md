> Этот документ предназначен для второго администратора (председателя ОСББ) и описывает текущее состояние системы, основные принципы её работы и ближайшие планы развития.

Документ не является окончательной документацией.

Система продолжает активно развиваться, поэтому руководство также будет регулярно дополняться новыми разделами.

Основная цель настоящего издания — помочь быстро разобраться в существующих возможностях системы и начать её практическое использование.

# Вместо предисловия

Практически любая информационная система начинается одинаково.

Сначала появляется программа.

Потом появляются пользователи.

И только после этого становится понятно, насколько удачными оказались принятые решения.

Наш проект развивается немного иначе.

Большая часть архитектуры, структуры базы данных и логики работы была разработана и многократно проверена ещё до появления первых реальных операторов.

В песочных базах данных были последовательно отработаны различные сценарии:

- регистрация жителей;
- привязка квартир;
- оформление заявок на услуги;
- контроль задолженности;
- оформление сервисных заказов;
- телефонный доступ к шлагбаумам;
- журналирование действий операторов;
- различные аварийные и конфликтные ситуации.

Только после успешного прохождения этих этапов система была перенесена в тренировочную базу данных, предназначенную для первого практического использования.

Сегодня начинается следующий этап.

Не разработка.

А опытная эксплуатация.

Именно на этом этапе особенно важен взгляд человека, который ежедневно будет пользоваться системой для решения реальных задач дома.

Поэтому замечания, вопросы и предложения второго администратора являются не менее важной частью проекта, чем программный код.
