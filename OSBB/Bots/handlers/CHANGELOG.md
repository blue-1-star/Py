# Vehicle Registry v0.2

- Добавлен жизненный цикл черновика автомобиля.
- Автомобиль можно создать без известной квартиры.
- Сохраняется полный номер или известный фрагмент, например `8092`.
- Черновик получает `review_status = NEEDS_REVIEW` и `plate_format_status = PARTIAL`.
- Поиск показывает автомобили без квартиры.
- Карточка явно помечает черновик.
- Квартиру можно добавить позднее через существующую карточку.
- После появления квартиры и полного украинского номера статус автоматически меняется на `VERIFIED`.
- `vehicle_id` остаётся прежним, поэтому будущие платежи и история не требуют переноса.
