# Changelog

## 1.0 — 2026-07-19

- Сохранены диагностические команды Assistant v0.3.
- Код разделён на модули без внешних зависимостей.
- Добавлена команда `assistant export resolver`.
- Реализован автоматический поиск SQLite-базы проекта.
- Экспортируются таблицы, столбцы, CREATE TABLE, внешние ключи, индексы, VIEW и триггеры.
- Формируется карта внешних ключей.
- В обычном режиме выделяются таблицы-кандидаты Resolver Layer.
- `--full` экспортирует все таблицы.
- Добавлены безопасные образцы данных с маскированием чувствительных полей.
- Отчёты записываются в `OSBB_util/Exports`.
